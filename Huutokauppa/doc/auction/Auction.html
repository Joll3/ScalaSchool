Auction

trait Auction extends AnyRef

Auction holds auctioned items that users can try to bid on until the auction item is closed.

Auction is a fairly easily extendable base for building an electronic auctionhouse. One of the key elements is the use of events, which allows for easy to plug in mixins that can register themselves as event listeners and react to what happens in the auction.

The dispatcher is provided in a separate mixin called Dispatcher. This allows substituting the mixin when testing or for example providing a dispatcher that can dispatch events to remote Listeners etc.
