  1. class BidBot extends Listener

    A BidBot has one owner and one auction that it must listen.

    A BidBot has one owner and one auction that it must listen. It can be set to bid for multiple auction items for the user. The bot will make new bids for an item in behalf of the user whenever another user makes a higher bit. However, the bot will never bid more on an item than the maximum bid ceiling set.
